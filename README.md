# 📘 Facebook UI Clone

A responsive **Facebook UI Clone** built using **HTML5** and **CSS3**. This project recreates the core user interface of Facebook, including the Login Page, Timeline, and Profile Page, to practice front-end web development skills.

---

## 📸 Project Preview

### Login Page
- Facebook-inspired login interface
- Email and password fields
- Create Account button
- Responsive layout

### Timeline Page
- Navigation bar
- Left and right sidebars
- Stories section
- News feed posts
- Contact list

### Profile Page
- Cover photo
- Profile picture
- User information
- Photo gallery
- Post section
- Like, Comment, and Share buttons
